# Luma Studio - Fase 1: Identidad visual

Fecha: 2026-06-18

## Proposito del sitio

Luma Studio sera un portafolio artistico administrable para mostrar fotografia, albumes, proyectos visuales y cortometrajes. La experiencia publica debe sentirse como una galeria editorial digital: visual, saturada, directa y cuidada, sin verse como pagina corporativa.

## Nombre provisional

Nombre de marca provisional: **Luma Studio**

Nombre de carpeta/proyecto: `luma`

Razon:

- Luma conecta con luz, imagen, fotografia y cine.
- Studio deja espacio para obra fotografica, audiovisual, editorial y experimental.
- Es corto, facil de pronunciar y no limita la marca a una sola disciplina.

## Personalidad visual

Palabras guia:

- Editorial
- Nocturna
- Saturada
- Cinematografica
- Curada
- Experimental
- Humana

La pagina debe iniciar mostrando obra, no explicando servicios. El primer impacto debe venir de imagen, movimiento y composicion.

## Direccion estetica

La direccion recomendada es **galeria nocturna editorial**:

- Fondos oscuros para elevar fotografias y stills.
- Acentos saturados para interacciones, botones y filtros.
- Layouts con composicion editorial: grillas asimetricas controladas, texto corto, mucho peso visual.
- Animacion sutil: aparicion, hover, transiciones de galeria y lightbox.
- Evitar tarjetas corporativas grandes, degradados genericos y secciones tipo SaaS.

## Paleta principal

### Galeria Nocturna

| Rol | Color | Uso |
| --- | --- | --- |
| Fondo base | `#0B0A12` | Fondo principal del sitio |
| Superficie | `#171426` | Paneles, admin, overlays |
| Superficie elevada | `#221B36` | Hover, modales, elementos activos |
| Texto principal | `#F6F1E8` | Titulos y texto sobre fondo oscuro |
| Texto secundario | `#AFA7BD` | Metadata, fechas, categorias |
| Acento fucsia | `#FF4D8D` | CTA principal, hover fuerte |
| Acento cian | `#00E0C6` | Filtros activos, links, detalle tecnico |
| Acento ambar | `#FFB000` | Estados destacados, chips |
| Acento verde | `#7CFF4B` | Microdetalles, tags experimentales |
| Borde sutil | `#342C49` | Separadores y contornos |

Regla de uso:

- 70% fondo oscuro y superficies.
- 20% imagen/obra visual.
- 10% acentos saturados.

No usar todos los acentos al mismo tiempo en una misma seccion. En cada pantalla debe dominar un acento y los demas quedar como apoyo.

## Tipografia

Propuesta inicial:

- Titulos: `Space Grotesk`, `Sora` o `Clash Display` si se licencia/disponibiliza correctamente.
- Texto UI: `Inter` o `Geist Sans`.
- Metadata editorial: la misma fuente UI con uppercase, tracking ligero y tamanos pequenos.

Jerarquia:

- H1 home: grande, corto, maximo 2 lineas.
- Titulos de proyecto: grandes, claros, con categoria visible.
- Texto descriptivo: parrafos breves de 2 a 4 lineas.
- Metadata: ano, tipo, cliente/rol, tags.

## Voz y contenido

Tono:

- Directo
- Artistico sin exageracion
- Curatorial
- Poco texto, muy visual

Microcopy recomendado:

- `Obra destacada`
- `Ver proyecto`
- `Explorar album`
- `Cortometrajes`
- `Archivo visual`
- `Publicado`
- `Borrador`

Evitar:

- "Soluciones integrales"
- "Nuestros servicios"
- "Innovacion y calidad"
- Lenguaje de agencia corporativa

## Principios de UI

- La obra siempre debe tener prioridad visual.
- Las imagenes deben tener dimensiones estables para evitar saltos de layout.
- La navegacion debe ser simple: Inicio, Proyectos, Albumes, Cortos, Sobre.
- El admin debe ser sobrio, mas funcional que expresivo.
- Toda galeria debe soportar portada, ordenamiento y estado publicado/borrador.
- Los filtros deben ayudar, no convertirse en obstaculo.

## Decisiones cerradas para fase 1

- Marca provisional: **Luma Studio**.
- Direccion visual: **galeria nocturna editorial**.
- Paleta base: **Galeria Nocturna**.
- Proyecto sera full-stack desde la primera implementacion.
- El contenido se modelara como proyectos, albumes y media assets.
- El panel admin sera privado y separado visualmente del sitio publico.
